- [ ] Python for AI (NumPy, Pandas, Matplotlib)

- [ ] Neural networks from scratch
- [ ] PyTorch fundamentals

- [ ] Transfer learning & fine-tuning

- [ ] How LLMs work (attention, tokenization, embeddings)
- [ ] Using OpenAI / Anthropic APIs

- [ ] LangChain fundamentals

- [ ] RAG pipeline design
- [ ] Document loaders & chunking strategies

## Stage 5 — AI Agents (Months 9–10)

- [ ] Agent frameworks (LangGraph, AutoGen)
- [ ] Tool use & function calling
- [ ] Memory systems (short-term, long-term)
- [ ] Multi-agent systems
- [ ] Building a production-ready agent

## Stage 6 — Production AI (Months 11–12)

- [ ] MLOps basics (model versioning, monitoring)
- [ ] FastAPI for AI endpoints
- [ ] Deployment (Docker, cloud)
- [ ] Cost optimization (caching, batching)
- [ ] Security & responsible AI
